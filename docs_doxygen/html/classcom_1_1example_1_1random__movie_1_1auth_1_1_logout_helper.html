Detailed Description

Вспомогательный класс для выхода пользователя из аккаунта.

Класс поддерживает:

  • выход только с текущего устройства;
  • выход со всех устройств пользователя.

В обоих случаях локальная сессия очищается независимо от результата сетевого запроса, чтобы обеспечить корректный logout на клиенте.
