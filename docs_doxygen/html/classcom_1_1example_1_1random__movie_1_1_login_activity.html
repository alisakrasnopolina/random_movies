Detailed Description

Экран входа пользователя в приложение.

Activity отвечает за:

  • ввод email и пароля;
  • локальную валидацию полей формы;
  • отправку запроса на backend для аутентификации;
  • сохранение access/refresh токенов и данных пользователя;
  • переход в основную часть приложения после успешного входа.

Выполняет вход пользователя через backend API.

Формирует JSON-запрос с email и паролем, отправляет POST-запрос на endpoint /auth/login, а при успешном ответе сохраняет токены и данные пользователя в SessionManager.

В случае ошибки сети показывает уведомление о проблеме с подключением. В случае ошибки сервера выводит сообщение, сопоставленное по HTTP-коду.

Обрабатывает ошибку сетевого запроса.

Parameters

Обрабатывает ответ backend после попытки входа.

При успешном ответе извлекает access token, refresh token и данные пользователя, сохраняет их локально и выполняет переход в основную часть приложения.
