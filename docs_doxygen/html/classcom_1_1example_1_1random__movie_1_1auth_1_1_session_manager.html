Detailed Description

Класс для локального хранения сессии пользователя.

SessionManager использует SharedPreferences для сохранения:

  • access token;
  • refresh token;
  • идентификатора пользователя;
  • email;
  • отображаемого имени;
  • флага принудительного выхода из аккаунта.
